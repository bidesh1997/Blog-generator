# Blog Generator with OpenAI API

## Project Description

This is a simple blog generator built using Python and the OpenAI API.
It allows users to input a topic and generates a paragraph
automatically.

## Requirements

-   Python 3.12 (Recommended)
-   Virtual Environment
-   openai
-   python-dotenv

## Installation

1.  Create and activate a virtual environment:

    python -m venv venv venv

2.  Install dependencies:

    pip install openai python-dotenv

## Environment Setup

Create a .env file in the project root:

API_KEY=your_api_key_here

## How to Run

Inside the activated virtual environment:

python main.py

## Features

-   Topic-based blog generation
-   Secure API key storage using .env
-   Clean CLI interface
